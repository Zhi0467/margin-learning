A color never arrives alone

The same swatch can feel warmer, cooler, brighter, or duller when only its surroundings change.

Look before naming

Both small squares use exactly the same yellow. Cover one half of the comparison, then switch. Notice which yellow seems warmer and which seems more luminous.

Retrieval
